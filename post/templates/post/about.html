The GRUG Stack

The GRUG stack is a hypermedia-driven application development environment simplifying the creation of web apps. It provides all the tools you need to build without having to rely on a frontend framework.

It utilizes HTMX and Hyperscript to create an interactice front end. These two open-source libaries allows you to access modern browser features directly from HTML to deal with AJAX requests, partial DOM updates, CSS transitions, event handling, Server-Sent Events and WebSockets. The core of this stack is Django, python web framework, which provides a secure and scalable backend with a template engine to render HTML. Ultimately, The GRUG stack aims to create web apps while keeping you in HTML and Django's Python code.
